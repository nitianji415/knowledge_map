# 贡献者行为准则 / Code of Conduct

## 我们的承诺

为营造开放、友善的社区,我们承诺让每个人参与本项目的体验都不受骚扰,无论年龄、体型、残障、族裔、性别认同与表达、经验水平、国籍、外貌、种族、宗教或性取向。

## 我们的标准

有助于营造积极环境的行为包括:

- 使用友善和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 以社区的整体利益为重

不可接受的行为包括:

- 骚扰、侮辱/贬损性评论,人身或政治攻击
- 公开或私下的骚扰
- 未经许可发布他人隐私信息
- 其他在专业场合中合理认定为不当的行为

## 执行

如遇违反行为准则的情况,可通过 GitHub 私下联系项目维护者举报。维护者有权且有责任移除、编辑或拒绝不符合本准则的评论、提交、代码、issue 等,并对其认为不当、威胁、冒犯或有害的行为做出处理。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) v2.1。
